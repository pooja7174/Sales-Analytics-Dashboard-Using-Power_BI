📊 Sales Analysis Dashboard using Power BI

📌 Project Overview

This project presents an interactive Sales Analysis Dashboard built using Power BI to analyze the performance of McDonald's menu items. The dashboard helps identify top-performing products, category-wise sales trends, and customer ordering patterns using data visualization and business intelligence techniques.

The objective of this project is to transform raw sales data into actionable insights that can support business decision-making, product strategy, and operational improvements.

The dashboard analyzes 5,000+ customer orders across 32 menu items and 10 product categories, providing a clear overview of sales distribution and product performance.

🎯 Project Objectives

Analyze overall sales performance of menu items

Identify top-selling and low-performing products

Understand category-wise contribution to total sales

Enable interactive exploration of sales data

Build a business-friendly dashboard for decision making

📊 Dashboard Features
1️⃣ KPI Overview

The dashboard includes key performance indicators that provide a quick summary of business performance:

Total Orders: 5K+

Total Categories: 10

Total Menu Items: 32

Total Sales: 158.09

These KPI cards allow users to quickly understand overall business performance at a glance.

2️⃣ Category-wise Sales Analysis

A bar chart visualization displays sales distribution across different food categories.

Categories include:

Burger

Chicken

Sandwich

Pasta

Wraps

Insights:

Burger category generated the highest sales

Wraps category showed the lowest sales contribution

This analysis helps identify which categories drive the most revenue.

3️⃣ Item-wise Sales Performance

This section highlights top-performing menu items based on total sales.

Example high-performing items:

McLobster Roll

Meatball Marinara

Angus Third Pounder

Chicken Alfredo Pasta

This helps businesses understand customer preferences and product demand.

4️⃣ Interactive Filtering (Slicers)

The dashboard includes interactive slicers that allow users to dynamically filter the data.

Available filters:

Category

Menu Items

Users can easily analyze sales for specific product groups or individual items.

5️⃣ Drill-through Analysis

A drill-through feature is implemented to enable deeper analysis.

For example:

Clicking on a category (e.g., Sandwich) opens a detailed page showing:

All items in that category

Individual item sales

Category total sales

This functionality provides multi-level data exploration.

6️⃣ Conditional Formatting

The sales table uses conditional formatting to visually highlight performance:

🟢 Green → High-performing items
🔴 Red → Low-performing items

This allows quick identification of strong and weak products.

7️⃣ Detailed Data Table

A detailed table provides granular insights including:

Category

Item Name

Total Sales

Total Orders

This enables users to perform item-level analysis.

🧹 Data Cleaning & Transformation

Data preprocessing was performed using Power Query.

Steps included:

Handling missing values

Structuring the dataset for efficient analysis

Several DAX measures were created to calculate key metrics:

Total Sales

Total Orders

Total Menu Items

These measures power the dynamic KPI cards and dashboard visuals.
